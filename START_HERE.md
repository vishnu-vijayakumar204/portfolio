## 🚀 3 Simple Steps to Get Started

### Step 1: Install (5 minutes)

Open Terminal and run:

```bash
cd /Users/vishnu.vijayakumar/Desktop/Vishnu/portfolio
npm install
```

This installs all the necessary packages.

### Step 2: Run Locally (1 minute)

```bash
npm run dev
```

Then open your browser to: **http://localhost:3000**

You should see your portfolio! 🎉

### Step 3: Customize (1-2 hours)

Now add your personal information. Open these files and update:

1. **`src/components/Hero.tsx`** - Your name, title, social links
2. **`src/components/Projects.tsx`** - Your freelance projects
3. **`src/components/Experience.tsx`** - Your work experience
4. **`src/components/Contact.tsx`** - Your email, phone

Each time you save a file, the browser will automatically refresh!
